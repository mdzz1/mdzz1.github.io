JSPDF + html2canvas + vue 导出PDF实现分页 页眉、页脚

发表于 2020-11-05 更新于 2020-11-09

场景

因为公司项目需要将整个页面导出成PDF文件并加上页眉、页脚

什么是 JSPDF ?

前端导出PDF的 js 库
基本用法

// 引入PDF
var doc = new jsPDF()
doc.text(20, 20, 'This is gray.')
pdf.save('name.pdf')

什么是 html2canvas ?

html2canvas 可以通过获取 HTML 的某个元素，然后生成 Canvas，从而让用户保存为图片。
html2canvas 工作原理是将当页面渲染成一个 Canvas 图片，通过读取 DOM 并将不同的样式应用到这些元素上。
html2canvas 不需要来自服务器任何渲染，整张图片都是在客户端浏览器创建。
基本用法

// html2canvas
html2canvas(element).then(canvas => {
  ...
})

html2canvas + jsPDF 实现页面导出分页PDF

主要原理：
通过html2canvas将页面解析成图片
然后通过 jsPDF.addImage方法 将图片写入PDF文件
废话不多说 直接上代码

import html2canvas from 'html2canvas'
import jsPDF from 'jspdf'

export const ExportPDF = (opts, callback) => {
    // 获取元素
    let element = document.querySelector(opts.id) 
    let w = element.offsetWidth 
    let h = element.offsetHeight
    let canvas = document.createElement('canvas')
    // 放大
    canvas.width = w * 20 // 将画布宽&&高放大两倍
    canvas.height = h * 20
    let context = canvas.getContext('2d')
    context.scale(20, 20)
    // 抗锯齿
    context.mozImageSmoothingEnabled = true
    context.webkitImageSmoothingEnabled = true
    context.msImageSmoothingEnabled = true
    context.imageSmoothingEnabled = true
    // 转化图片
    html2canvas(element).then(canvas => {
      // A4纸，纵向
      var pdf = new jsPDF('p', 'mm', 'a4')
      var ctx = canvas.getContext('2d'),
          a4w = 200, a4h = 257, // A4大小，210mm x 297mm，左右保留5mm，上下保留20mm的边距，显示区域200 x 257
          imgHeight = Math.floor(a4h * canvas.width / a4w), // 按A4显示比例换算一页图像的像素高度
          renderedHeight = 0
      let logo = document.getElementById('PDF_LOGO')
      while(renderedHeight < canvas.height) {
        let page = document.createElement('canvas')
        page.width = canvas.width
        page.height = Math.min(imgHeight, canvas.height - renderedHeight)
        // 用getImageData剪裁指定区域，并画到前面创建的canvas对象中
        page.getContext('2d').putImageData(ctx.getImageData(0, renderedHeight, canvas.width, Math.min(imgHeight, canvas.height - renderedHeight)), 0, 0)
        // 添加图像到页面，保留5mm/20mm边距 4为了调整图片居中 按需自行调整
        pdf.addImage(page.toDataURL('image/jpeg', 1.0), 'JPEG', 4, 20, a4w, Math.min(a4h, a4w * page.height / page.width)) 
        // 分页处理
        renderedHeight += imgHeight
        if(renderedHeight < canvas.height) pdf.addPage()
      }
      pdf.save(opts.name)
      // 回调
      callback && callback()
  })
}

但是中文直接乱码了！！！（大坑） 查了下是JSPDF不支持UTF-8 丢了

JSPDF 支持中文（UTF-8）

没办法直接继续找解决办法 就是设置中文字体 

pdf.addFileToVFS('NotoSansCJKtc-Regular-normal.ttf', font())
pdf.addFont('NotoSansCJKtc-Regular-normal.ttf', 'NotoSansCJKtc-Regular', 'normal')
pdf.setFont('NotoSansCJKtc-Regular')

这样字体就OK了 这里font是字体文件 base64 字符串通过这里 将你想要的字体文件转换成js 我们只要里面的base64 字符串
(有钱提示：我试了好几个字体 都不行 就上面这个可以)

let index = 0
const GetHeight = (ele, height) => {
  // 高度
  let h = 0
  // 获取子元素
  let child = $(ele).children('div:visible')
  for(let i = index; i < child.length; i++) {
    if(h + $(child[i])[0].offsetHeight > height) {
      index = i
      break
    } else {
      h += ($(child[i])[0].offsetHeight + 20)
    }
  }
  return h
}

黑背景

但我处理好截断问题后 发现有背景黑块

处理方法如下：

// 白色填充（去除黑色背景）
let context = page.getContext('2d')
context.fillStyle = '#FFF'

Vue.extend 你学废了吗？

发表于 2020-11-02 更新于 2020-11-05

什么是 extend ?

extend是VUE的API
用法：使用基础 Vue 构造器，创建一个“子类”。参数是一个包含组件选项的对象。
data 选项是特例，需要注意 - 在 Vue.extend() 中它必须是函数 与 组件内data相同

<div id='app'>
  <span>MyApp</span>
</div>
<script>
  var Profile = Vue.extend({
    template: '<p>目标：绑定到id为app的元素下面  打印data中的数据,</p>'，
    data: function(){
      return {
          p1:'打印记录1',
          p2:'打印记录2'
        }
      }
  })
  // 进行绑定
  new Profile().$mount('#mount-point');
</script>

为什么要使用 extend ?

比如系统要做个全局信息提示组件，这时候用components会很麻烦 需要单独的属性去控制显示隐藏会很累赘
这个时候就可以用到 extend

一 构造一个自定义弹窗

<template>
   <div v-if="show" ref="modal" class="ek-modal_wrap">
     <div class="ek-modal-content">
       <div class="modal-title-wrap">
         <div class="modal-title">Vue.extend 你学废了吗？</div>
         <slot name="description"></slot>
       </div>
       <div class="modal-button">
         <a v-if="confirmVisible" class="contral-btn confirm-btn" href="javascript:;" @click="confirm"></a>
         <a v-if="cancleVisible" class="contral-btn cancle-btn" href="javascript:;" @click="cancle"></a>
       </div>
     </div>
   </div>
</template>

<script>
 export default {
   data() {
     return {
       show: true,
       title: '', // 标题
       confirmText: '确定', // 确认文字
       confirmVisible: true, // 是否展示确认按钮
       onConfirm: () => { // 确认执行函数
         this.$emit('confirm')
       }, 
       cancleText: '取消', // 取消文字
       cancleVisible: true, // 是否展示取消按钮
       onCancle: () => { // 取消执行函数
         this.$emit('cancle')
       } 
     }
   },
   methods: {
     confirm() {
       this.onConfirm()
       this.close()
     },
     cancle() {
       this.onCancle()
       this.close()
     },
     close() {
       this.show= false
       if (this.$refs.modal) {
         this.$refs.modal.remove() // 关闭时候直接移除当前元素
       }
     }
   }
 }
 </script>

二 将方法绑定在 Vue 原型上

import Vue from 'vue'
import dialog from './components/Dialog.vue'
function showDialog(options) {
  const Dialog = Vue.extend(dialog) // 返回一个vue子类
  //创建实例并且挂载
  const app = new Dialog().$mount(document.createElement('div'))
  //初始化参数
  for (let key in options) {
    app[key] = options[key]
  }
  //将元素插入body中
  document.body.appendChild(app.$el)
}
Vue.prototype.$showDialog = showDialog //将方法放在原型上。
